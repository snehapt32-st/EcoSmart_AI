# EcoSmart AI – Climate Action Web Application

## Overview

EcoSmart AI is a single-page web application developed to support **Sustainable Development Goal 13 (Climate Action)**. The application is designed to promote environmental awareness and encourage sustainable living practices through an interactive and user-friendly interface.

The system integrates multiple features, including an eco-tips generator, a carbon footprint calculator, and a rule-based AI assistant, all implemented using basic web technologies without external dependencies.

---

## Objectives

* To raise awareness about climate change and its impact
* To encourage users to adopt eco-friendly habits
* To provide a simple tool for estimating individual carbon footprint
* To demonstrate the use of AI-inspired logic in a lightweight web application

---

## Features

### 1. Home Section

* Displays the application title and introduction
* Provides a brief overview of climate awareness
* Clean and structured layout for user engagement

### 2. Eco Tips Generator

* Generates 3–5 eco-friendly suggestions dynamically
* Covers areas such as:

  * Energy conservation
  * Waste management
  * Sustainable transportation
* Designed using simple logic to simulate AI-based responses

### 3. Carbon Footprint Calculator

* Accepts user inputs:

  * Daily travel distance (in kilometers)
  * Electricity usage (in hours)
  * Waste generation level (Low, Medium, High)
* Processes inputs to estimate environmental impact
* Displays results categorized as:

  * Low Impact
  * Medium Impact
  * High Impact
* Visual feedback through color-coded output

### 4. AI Chat Assistant

* Interactive input-output interface
* Provides responses to environmental queries such as:

  * Methods to reduce pollution
  * Energy-saving techniques
  * Sustainable lifestyle suggestions
* Uses predefined logic to simulate conversational AI behavior
* Includes greeting functionality for initial interaction

### 5. Climate Awareness Section

* Provides static informational content on:

  * Climate change
  * Importance of environmental protection
  * Overview of SDG 13
* Organized using structured sections for readability
